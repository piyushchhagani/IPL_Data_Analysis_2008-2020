# IPL Data Analysis Dashboard (Power BI)

## 📊 Overview
This project analyzes Indian Premier League (IPL) data using Power BI to generate meaningful insights related to team performance, player statistics, and match trends.

## 🎯 Objectives
- Analyze team performance across seasons  
- Identify top batsmen and bowlers  
- Study match trends and scoring patterns  
- Evaluate toss impact on match outcomes  
- Analyze scoring patterns across match phases  

## 📁 Dataset
The dataset includes:
- Matches data (teams, winners, venues, toss)
- Deliveries data (runs, wickets, players)

## ⚙️ Tools Used
- Microsoft Power BI Desktop  
- CSV Dataset  

## 📈 Dashboard Features
- KPI Cards (Total Matches, Runs, Avg Runs, Wickets)  
- Team performance analysis  
- Top batsmen and bowlers  
- Toss impact analysis  
- Over phase analysis  

## 🔍 Key Insights
- Mumbai Indians is the most successful team  
- Death overs contribute significant runs  
- Toss impact is moderate  
- Top players show consistent performance
